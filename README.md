# VibeFruit 🍓

En modern webbsida för VibeFruit - en frukt- och Nutella-butik.

## Om projektet

VibeFruit är en webbsida som showcasar våra unika frukt- och Nutella-kombinationer. Webbsidan är byggd med HTML, CSS och vanilla JavaScript.

## Funktioner

- 🎨 Modern och responsiv design
- 📱 Mobil-vänlig layout
- ✨ Smooth scroll-animationer
- 🖼️ Galleri med produkter
- 📋 Meny med priser
- 👥 Om oss-sektion
- 📞 Kontaktinformation

## Filer

- `index.html` - HTML-struktur
- `style.css` - CSS-styling
- `script.js` - JavaScript för animationer

## Hur man kör

1. Klona detta repository
2. Öppna `index.html` i din webbläsare
3. Eller använd GitHub Pages för hosting
